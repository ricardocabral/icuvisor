# v0.5 internal beta operator pack

Use these artifacts to run the maintainer-led v0.5 beta later. This folder is prep only: do not recruit, onboard, run interviews, or fill findings as part of TP-041.

## Execution order

1. [Protocol](protocol.md) — recruit 5-10 eligible participants within a 14-day window, collect consent, and screen fit.
2. [Onboarding playbook](onboarding-playbook.md) — install the signed macOS DMG, run setup, connect an MCP client, and verify the first call.
3. [Measurement plan](measurement.md) — collect KR1 and PRD §7.4 #6/#8/#12 evidence manually during the run.
4. [Exit interview](exit-interview.md) — run the end-of-beta script after each participant completes the trial.
5. [Findings template](findings.md) — fill one row per participant during/after the run; it is intentionally empty now.
6. [Operator checklist](checklist.md) — single-page Recruit → Onboard → Run → Synthesize flow.

## Privacy rule of thumb

Collect operational metadata and qualitative feedback only. Do not ask participants to send raw training data, API keys, athlete IDs, tool arguments, payloads, transcripts, or screenshots containing values.
